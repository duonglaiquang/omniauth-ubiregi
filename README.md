Omniauth for [Ubiregi](https://ubiregi.com/)

Instration
----------

Add this line to your application's Gemfile:

    gem 'omniauth-ubiregi'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install omniauth-ubiregi

Test
----

```
bundle exec rspec
```
